# 🚫 Fake Window.open Blocker

Un userscript simple et efficace pour bloquer les popups publicitaires tout en évitant de casser le fonctionnement des sites.

## ✨ Fonctionnalités

* ❌ Bloque toutes les tentatives de `window.open`
* 🧠 Simule un objet `window` valide (évite les erreurs JS)
* ⚡ Ultra léger et rapide
* 🔧 Compatible avec Tampermonkey / Violentmonkey

---

## 🔄 Installation (avec mise à jour automatique)

1. Installe un gestionnaire de userscripts :

   * Tampermonkey
   * Violentmonkey

2. Ajoute le script via une URL brute GitHub :

```
https://github.com/iokza/NoAds4Website/raw/main/script.user.js
```

👉 Remplace par ton propre lien une fois uploadé.

---

## 🔁 Grâce à ce Repo Github, le script restera à jour sur vos sites favoris

```javascript
// @updateURL   https://github.com/iokza/NoAds4Website/raw/main/script.user.js
// @downloadURL https://github.com/iokza/NoAds4Website/raw/main/script.user.js
```

💡 Cela permet à Tampermonkey de vérifier les mises à jour automatiquement.

---

## ⚠️ Disclaimer

Ce script est destiné à améliorer l'expérience utilisateur.
Certains sites peuvent ne pas fonctionner correctement si `window.open` est requis pour des fonctionnalités légitimes.
